Change List

2018.6.0

  • Introduce a Project Structure dialog, available from the File menu. This allows changing the project's Perl 6 SDK, as well as editing various elements of META6.json.
  • Provide support for editing module dependencies in META6.json in the new Project Structure dialog, with auto-complete of distribution names based on data from the module ecosystem.
  • Implement basic Pod parsing (recognize the block, paragraph, and abbreviated forms), with highlighting of directives and text
  • Support =finish and its various other forms
  • When the current file has a =finish, suggest $=finish in auto-complete
  • When $=finish is used without a =finish section, mark it as a problem
  • Recognize .pod6 files as Perl 6 and highlight them
  • Recognize halfwidth corner brackets in regexes
  • Enable the "Favorites" panel (used to throw an exception on opening)
  • Fix parser to accept whitespace after a prefix operator
  • Fix parser to accept a circumfix after an is trait longname
  • Fix %.foo{bar} parsing crash
  • Disable SSL verification when submitting bug reports, in an effort to make it work for those behind coroproate firewalls
  • Fix ModuleType exception that was thrown when creating a new project or doing a project import
  • Treat all files under bin/ as Perl 6 scripts
  • Fix multiple Unbalanced Tree parser failures
  • Fix indentation after a unit class declaration
  • Make auto-insertion of a closing brace work at the very end of the file
  • Avoid an infinite recursion when resolving module symbols
  • Fix an off-by-one in translating between IDEA and MoarVM line numbers in the debugger
  • Improve file placement for File > New > Module/Test to place under lib and t respectively
  • Provide a Send Feedback link so that the Send Feedback menu item now works
