Change List

2018.9.0

  • Implement Find Usages feature for variables, attributes, subs, private methods, and types
  • Implement Rename refactoring for variables, attributes, subs, private methods and types (the type case has some known limitations with regards to nested packages, which will be resolved in a future release)
  • Implement spell checking in comments and Pod
  • Parse and highlight the also keyword, and respect the traits that it applies to a package
  • Parse and highlight the trusts keyword, and allow auto-complete and reference resolution of private method calls allowed because of its use
  • Implement inserting #| on next line when pressing enter, to allow easier continuation of writing documentation comments
  • Provide better default run configuration names
  • Fix assorted issues with symbol resolution in more complex cases of class inheritance and role composition structures
  • Fix auto-complete of $.foo style calls
  • When the project SDK changes, invalidate cached symbol resolutions based on the previous SDK
  • Fix exceptions on empty project creation
  • Fix exception when META6.json is invalid
  • Fix exception when subset keyword is not followed by a name and indexing takes place
